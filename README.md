# M5EPD-TEMPLOG
気温と湿度をSDカードに記録する。10分のRTC起動のインターバル設定。トレンドグラフを表示するために内蔵I2C経由でEEPROMに過去のデータを毎回記録および呼び出しを行っている。<BR>
SDカード上にフォントファイル"/fonts/ipaexg.ttf"が必要。<BR>
  10分インターバル設定で十日間稼働し電圧が4394から3732まで低下3608<BR>
  HOMEKITへの対応を予定<BR>
